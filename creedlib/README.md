# CreedLib

**CreedLib** is a lightweight, standalone library designed for use in FiveM scripts. It provides a collection of utility functions and core features to simplify development while maintaining compatibility with ox_lib and other standalone resources. 

---

## Features
- **Client Functions**: Utility functions like player teleportation, notifications, and walking style changes.
- **Server Functions**: Utilities for logging, player management, and server-wide announcements.
- **Shared Utilities**: A collection of shared functions for data manipulation, debugging, and formatting.
- **Standalone**: No framework required, but integrates seamlessly with ox_lib for notifications and utilities.
- **Export-Based API**: Access functions via exports, similar to popular frameworks like qb-core.
